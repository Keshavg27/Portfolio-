# Portfolio Starter

This is a simple personal portfolio website starter built with plain HTML, CSS, and JavaScript.

## Files

- `index.html`: Main page structure and content
- `styles.css`: Visual design and responsive layout
- `script.js`: Small scroll reveal animation

## What to edit first

Replace the placeholder content in `index.html`:

- Your intro text
- College / university and degree details
- Work or internship experience
- Project names and descriptions
- Skills
- Email, LinkedIn, and GitHub links

## Preview locally

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
